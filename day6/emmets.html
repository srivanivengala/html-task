Discription
Child: >

Explanation: An element that is directly below and connected to an element in the document tree.

Example: html>head>title

          < html >
                 < head >
                 < title >< /title >
                 < /head >
            < /html >
sibling: +
 Explanation: Shares same parent with the element

Example: div+p+bq

           < div>< /div>
             < p>< /p>
             < blockquote>< /blockquote>
Climb-up: ^ Explanation: It is used to moves up one level in hierarchy,allowing us to add sibling elements to the parent elements

Example : div+div>p>span+em^bq 
            < div>< /div>
                < div>
                    < p>< span>< /span>< em>< /em p>
                    < blockquote>< /blockquote>
                < /div>
Example : div+div>p>span+em^^bq 
                < div> < /div>
                < div>
                     < p>< span>< /span>< em>< /em>< /p>
                < /div>
                < blockquote>< /blockquote>
Multiplication:*

Explanation: It is used to get multiple elements

Example : ul>li*5 
         < ul>
                < li>< /li>
                < li>< /li>
                < li>< /li>
                < li>< /li>
                < li>< /li>
            < /ul>
Item numbering: $ Example: ul>li.item$*5 
                < ul>
                    < li class="item1">< /li>
                    < li class="item2">< /li>
                    < li class="item3">< /li>
                    < li class="item4">< /li>
                    < li class="item5">< /li>
                < /ul>
Grouping: () Example : div>(header>ul>li*2>a)+footer>p 
       < div>
            < header>
                < ul>
                    < li>< a href="">< /a>< /li>
                    < li>< a href="">< /a>< /li>
                < /ul>
            < /header>
            < footer>
                < p>< /p>
            < /footer>
        < /div>
Text: {} To create text with in the element
Example : a{Click me} 
< a href="">Click me< /a>
{Click }+a{here}+{ to continue} 
< p>Click < a href="">here< /a> to continue< /p>
Custom attributes: [] To show custom attributes
Example : td[rowspan=2 colspan=3 title] 
< td rowspan="2" colspan="3" title="">< /td>
