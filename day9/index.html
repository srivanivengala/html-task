HTML

Types of Tags
  • Container Tag(or)Pair Tag: Tags which contains opening tag and closing tag
    • Syntax: < opening tag >< /opening tag >
  • Empty Tag (or)Void Tag: Tags which doesnot require closing tags
    • Syntax:< Tagname >
Element

  • Inline Element: Elements which doesnot starts on new line and which takes necessary place only. Examples: Anchor tag, span tag etc..
  • < Opening tag >some content < /closing Tag >

Arrtibutes of the table
